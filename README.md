# Projeto Ebook Gerado por I.A.s
O projeto tem o objetivo de gerar um Ebook utilizando ferramentas de IA através de prompts mais trabalhados.  
Utilizar uma esteira de prompts para gerar cada etapa do processo criativo e de desenvolvimento do Ebook.

## Tecnologias utilizadas para o projeto
* [ChatGPT](https://chatgpt.com/) - Titulo e capitulos do Ebook  
* [Stable Diffusion](https://stabledifffusion.com/tools/ai-image-generator#google_vignette) - Capa  
* [Canvas](https://www.canva.com/) - Criação e edição do PDF
