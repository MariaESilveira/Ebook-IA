# Prompt para geração dos capitulos do Ebook

Faça um texto para ebook, sobre uma história de aventura em um mundo que retrata a programação, com uma heroína que surge em um mundo desconhecido e busca superar os desafios e desvendar o mundo dos códigos. Cada capitulo deve conter uma pequena história com essa heroína, que trate dela resolvendo algum desafio e aprendendo com a programação de códigos.   

## REGRAS  

* Fale sempre de uma maneira envolvente e simples  
* Explique termos técnicos de forma simples e envolvente   
* Deixe o texto enxuto    
* Crie um nome para cada capitulo
